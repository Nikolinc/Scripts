# Windows Update Reset Script

## Описание
Этот скрипт PowerShell полностью снимает ограничения и блокировки на обновления Windows 11. Он:

- Включает все необходимые службы Windows Update (`wuauserv`, `BITS`, `CryptSvc`, `TrustedInstaller`).
- Удаляет политики и реестр, которые блокируют автоматические обновления.
- Очищает кэш и временные файлы обновлений (`SoftwareDistribution`, `catroot2`).
- Восстанавливает повреждённые системные компоненты с помощью `DISM` и `SFC`.

После выполнения система будет готова для нормального получения и установки обновлений.

---

## Предварительные требования
- Windows 11
- Запуск скрипта **от имени администратора**
- PowerShell версии 5.1+

> ⚠️ Внимание: скрипт изменяет реестр и системные файлы. Рекомендуется создать точку восстановления перед использованием.

---

## Использование
1. Скопируйте скрипт в файл, например `Reset-WindowsUpdate.ps1`.
2. Запустите PowerShell от имени администратора.
3. Разрешите выполнение скриптов, если необходимо:
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
4.	Запустите скрипт:

```powershell
.\Reset-WindowsUpdate.ps1
```

5.	Дождитесь окончания выполнения.

6.	Перезагрузите компьютер.
